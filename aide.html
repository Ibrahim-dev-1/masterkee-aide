Préréquis

Pour vous s'inscrire à MasterkeeGroup, vous devez payer une adhésion de 10 000 fcfa via Flooz, Tmoney, ou Xpress sur le numero: numéro masterkeegroup

Une fois l'adhésion payée, vous aurez automatiquement un numéro d'inscription, que vous aurez bésoins pour vous inscrire.

NB: Il est vivement conseillez d'utiliser son propre numéro pour l'envoie de l'argent.

Etape 1

* Cliquez sur le bouton s' inscrire en haut de la page, comme le montre l'image.

Etape 2

1- Cliquez dans le champ numéro 1, puis entrez le code d'inscription que vous avez reçus après l'envoie de votre argent

2- En suite vous cliquez dans le champ numéro 2, puis entrez le code de votre commissionnaire (votre Parain )

3- Entrez votre Nom de famille dans le champ numéro 3

4- Entrez votre Prénom dans le champ numéro 4

5- Entrez votre Email dans le champ numéro 5

NB: Assurez-vous d'avoir entrez une addresse email valid , car vos informations de connexion, vous seront envoyés dessus.

6- Continuez en sélectionnent votre Pays dans le champ 6

7- choisissez votre ville

8- Maintenant, entrez dans les champs 8 et 9 vos deux numeros de téléphone (moov et Togocel) si vous en avez. Au cas contraire, entrez le meme numéro dans les deux champs

Etape 4

2- Une fois information Mémorisez , cliquez sur le boutton Acceder au BackOffice (votre compte)
